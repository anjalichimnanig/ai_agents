# AI Agents

## Learning, Developing and Deploying AI Agents for Enterprises 
AI Agents are systems that perform tasks autonomously where they plan, reason, act, and adapt to achieve pre-determined goals. These systems are proactive and take decisions to complete the tasks specified. 
The repository contains and helps to build these Agents with growing level of complexity. 

* Hello World
AI Agent to Get Started. Help to answer any user query based on its own knowledge. \
    * Agent - root_agent
    * Model - gemini-2.5-flash
    * Tool - No Tool

* Bank Customer Service
AI Agent to respond to user queries based on an external static FAQ provided. The Agent responses are grounded on the knowledge provided.\
    * Agent - customer_service_representative
    * Model - gemini-2.5-flash
    * Tool - get_bank_information
